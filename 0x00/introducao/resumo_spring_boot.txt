	O que é Spring Boot Framework?
É uma ferramenta que auxilia a criar aplicações standalone (funcionam por si própria), ajuda nas configurações iniciais do nosso projeto.
     Quais são os benefícios e recursos que Spring Boot fornece?
O spring boot, facilita os processos de aplicações java, traz agilidade para o processo de desenvolvimento, ajuda nas configurações iniciais, permite ter um projeto mais organizado.
     Como é feita a adição de novas dependências (JPA, Hibernate, etc…) ?
Para adição de dependências pode-se usar o contêiner IoC esse container gerencia as dependências do projeto de forma automatica, esse objetos gerenciados são chamados de beans. (ex: @Component @Service @Repository etc).

     O que é Spring Initializr?
O Spring initializr para mim, é o Santo Graau, gosto muito e é muito fácil de usar! 
Esse Spring initializr nada mais é do que uma interface “amigável” web, onde o programador consegue gerar seu projeto inicial a partir dali. Onde pode-se colocar configurações pré-moldadas. Facilitando assim a nossa vida.
